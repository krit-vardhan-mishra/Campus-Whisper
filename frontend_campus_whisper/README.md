# Frontend Campus Whisper

This is a standalone React Frontend application built with Vite and TypeScript.

Live demo: https://campus-whisper.onrender.com

## Getting Started

### Prerequisites
*   Node.js installed

### Installation

1.  Navigate to the project directory:
    ```bash
    cd frontend_campus_whisper
    # (Or if you are already in the directory)
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Configuration

*   Check `.env.local` for environment variables (e.g., `MONGODB_URI` or API details).
    *   *Note: This project is configured with a MongoDB URI for potential future backend integration.*

### Running the Application

*   **Start Development Server:**
    ```bash
    npm run dev
    ```
    This will start the Vite development server (usually on port `5173`).

*   **Build for Production:**
    ```bash
    npm run build
    ```

*   **Preview Production Build:**
    ```bash
    npm run preview
    ```
